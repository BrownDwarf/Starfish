Is the noise in a spectrum correlated?

We seek to characterize the covariance in our spectrum and understand its origin. In order to do this, we use high signal to noise TRES observations of Vega.

Synopsis

The "global covariance structure" is not due to any correlation in the actual physical noise itself. We verify this by measuring the autocorrelation of regions of stellar continuum and show that there is no significant correlation.

However, there is indeed a global covariance structure present in the residuals of any spectroscopic fit. This is because the model is never a perfect fit to the data. If the model is a perfect fit to the data (as in the case of stellar continuum), then the residuals will be uncorrelated. However, most spectral lines (even the best modeled ones) are either under-predicted or over-predicted and thus result in correlated residuals.

This is borne out by the fact that the global covariance for the Kurucz spectra (better models) is lower in amplitude and characteristic length than the PHOENIX spectra.

To test whether or not there is correlation, we will calculate an estimate of the autocorrelation

\[\hat{R}(k) = \frac{1}{(n - k) \sigma^2} \sum_{t = 1}^{n - k} (X_t - \mu) (X_{t+k} - \mu) \]

def estimated_autocorrelation(x):
    n = len(x)
    variance = np.var(x)
    x = x - np.mean(x)
    r = np.correlate(x, x, mode = 'full')[-n:]
    #assert N.allclose(r, N.array([(x[:n-k]*x[-(n-k):]).sum() for k in range(n)]))
    result = r/(variance*(np.arange(n, 0, -1)))
    return result

Interestingly Vega 1 looks like there is some correlation on ~5 pixel scales, but this does not appear in Vega 2. This looks like a coincidence.

To check our assertion that noise in the continuum is actually uncorrelated, let's see what the autocorrelation of correlated noise actually looks like. We will generate noise with a known correlation length.

#Try generating correlated noise using the Matern kernel

def k_3_2(r, l):
    return (1 + np.sqrt(3) * r/l) * np.exp(-np.sqrt(3) * r/l)

@np.vectorize
def hann(r, r0):
    if np.abs(r) < r0:
        return 0.5 + 0.5 * np.cos(np.pi * r/r0)
    else:
        return 0

def k_3_2_hann(r, l):
    return k_3_2(r, l) * hann(r, 6 * l)

def k_3_2_hann_func(x0i, x1i, x0v=None, x1v=None, amp=None, l=None):
    x0 = x0v[x0i]
    x1 = x1v[x1i]
    r = np.abs(x1 - x0)
    return amp**2 * k_3_2_hann(r, l)

If this is the case, and true spectroscopic noise is uncorrelated, then where does the global covariance structure come from in our fit? Let's test this by examining actual residuals from a fit to WASP14 using the PHOENIX spectra. I have chosen a region of the spectrum that has a few spectral lines but does not have any egregious lines that would be marked by a "bad region."

Boom, the autocorrelation shows that the residual pixels are indeed correlated on the ~few pixel length scale, similar to what we saw with the forcibly correlated noise.

These residuals are correlated due to slight mismatches in the line shapes between model and data spectrum. This explains why the covariance fit for the Kurucz spectra has a smaller amplitude and length scale than the PHOENIX spectra. If the spectral models fit better (Kurucz) then there will be less residual structure.
